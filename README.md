# 👟 ShoeStore – Frontend

ShoeStore is a modern and responsive e-commerce frontend built using **React.js**.  
---

## 🚀 Features

- Responsive UI
- Product listing with categories
- Search products by name / category
- Product detail page
- Clean and modern design
- Fast performance with React
- User-friendly navigation

---

## 🛠️ Technologies Used

- React.js
- React Router DOM
- JavaScript (ES6+)
- Tailwind CSS
- HTML5
- CSS3
